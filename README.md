## Setup
To run this project, install it locally using npm 
### SERVER:

```
$ cd ../MERN-Blog/API
$ npm install
$ npm start
```

### CLIENT:
```
$ cd ../MERN-Blog/front
$ npm install
$ npm start
```

You have to also create .env file in both directories with yours credentials.
